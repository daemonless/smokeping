## Deployment

### Podman Compose

```yaml
services:
  smokeping:
    image: ghcr.io/daemonless/smokeping:latest
    container_name: smokeping
    environment:
      - PUID=1000
      - PGID=1000
      - TZ=UTC
    volumes:
      - /path/to/containers/smokeping:/config
      - /path/to/data:/data
    ports:
      - 8081:8081
    restart: unless-stopped
```

### Podman CLI

```bash
podman run -d --name smokeping \
  -p 8081:8081 \
  -e PUID=@PUID@ \
  -e PGID=@PGID@ \
  -e TZ=@TZ@ \
```
